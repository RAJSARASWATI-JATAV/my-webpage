## 📦 Installation

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Setup

```bash
# Clone the repository
git clone <repository-url>
cd my-pc

# Install dependencies
npm install

# Start development server
npm run dev
```

Open your browser and navigate to `http://localhost:5173`

## 🔧 Customization

### Colors
Edit `src/index.css` to modify color variables:
```css
--neon-green: #00ff41;
--neon-cyan: #00ffff;
--accent-purple: #b000ff;
```

### Animations
Add custom animations in `src/index.css`:
```css
@utility animate-custom {
  animation: custom 2s ease-in-out infinite;
}
```

### Components
All components are in `src/components/` - fully customizable and reusable.
